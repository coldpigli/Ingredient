Avatars

Avatars are images that users can set as their profile picture. On Ingredients, they're always going to be rounded or rounded-squares. They can be custom photos, uploaded by users, or generated as identicons as a placeholder.

The Avatar component is used to represent users. And should only be used for users.

Types of Alerts in Ingredients

We at Ingredients know that you love choices. Sweat not! We've got you covered. Avatars in Ingredients come in 3 main variants. They are:

  • Default Rounded Avatars
  • Avatars with different dimensions
  • Squircle Avatar

Let's look at how these variations will look at your screen.

Default Rounded Avatar

A rouned avatar with dimensions "to be added" is the default avatar in Ingredients. It can be tapped by using the "avatar" class on your "img" tags.
