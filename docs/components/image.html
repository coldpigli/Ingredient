Images

Modern day websites are incomplete without images. You'll need images in one form or the other. With different screen sizes we have today, an image needs to be responsive and adjust their size according to the screen size.

Responsive Image Components are used to dsiplay Images on websites, which are screen responsive. They expand upto the width of the container they are in.

Types of Images.

We at Ingredients know that you love choices. Sweat not! We've got you covered. Images in Ingredients come in 3 main variants. They are:

  • Responsive Images
  • Square Images
  • Round Images
